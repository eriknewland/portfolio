AI-RPG

AI-RPG is a live iOS application available for download in the Apple App Store.

Apple App Store Link

Overview

A modern twist on the classic text-adventure RPG. OpenAI's API powers this React Native iOS application to create a unique adventure every play through.

Users experience a world of limitless adventure as they explore a new take on a classic game. This game was made entirely with React Native and Expo without Xcode. This application was very difficult to debug without Xcode, but hardware constraints kept me from using Xcode for this application. Notably, advertisements and in-app payments were very difficult to implement, as well as some polyfills to enable response streaming from my axios calls.
